# Dashboard

The dashboard is your home screen when you log into Therativities. It provides a snapshot of your practice activity.

## What you'll see

- **Upcoming sessions** — Your next scheduled appointments
- **Recent activity** — Latest updates across your practice
- **Key metrics** — Session counts, revenue, and client stats
