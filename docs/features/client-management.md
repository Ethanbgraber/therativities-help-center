# Client Management

Therativities keeps all your client information organized in one place — profiles, guardians, sessions, goals, billing, and documents.

## Adding a client

1. Click **Add New Client** in the sidebar (or the floating action button on mobile)
2. Fill in the required fields:
   - **First Name** (required)
   - **Last Name** (required)
   - **Sex** (required — Male or Female)
   - **Date of Birth** (age is calculated automatically)
3. Click **Create Client**
4. You'll be asked: **Onboard Now** or **Skip For Now**
   - **Onboard Now** takes you to the guardian info step to add parent/guardian details
   - **Skip For Now** creates the client and you can add details later

## Client list

Your client list shows all active clients in a table with these columns:

| Column | Description |
|--------|-------------|
| Client Name | Sortable by first or last name |
| Next Session | Date and time of the next scheduled session (clickable) |
| Last Session | Most recent session (clickable) |
| Past Sessions | Count of sessions needing review |
| Status | Current status badge |
| Actions | Quick action menu |

### Status badges

- **In Session** (green) — A session is currently in progress
- **Scheduled** (blue) — Has an upcoming session
- **Not Scheduled** (gray) — No upcoming sessions
- **Start Onboarding** / **Finish Onboarding** (purple) — Client setup is incomplete

### Filtering and sorting

- **Search** — Filter by client name or parent name
- **Sort by** — Next Session, Last Session, Past Sessions, First Name, Last Name, or Status
- **Show Archived** — Toggle to include archived clients

### Quick actions

From the actions menu on each client row:
- **Plan Session** — Jump to the next upcoming session
- **Process Billing** — Go to billing for the last session
- **View Details** — Open the full client profile
- **Archive Client** — Remove from active list (can be unarchived later)

## Client profile

Click a client to open their full profile. The profile is organized into tabs:

### Overview
A dashboard for this client with key stats, recent sessions, and a goals summary.

### Sessions & Schedule
View all past and upcoming sessions. Book new sessions, manage recurring schedules, and review session history.

### Goals & Treatment
Set long-term goals, short-term objectives, and track measurable progress. Use the **Brain Dump** feature to speak or type unstructured observations and let AI structure them into goals.

### Evaluations
Create and manage evaluation reports. Upload clinical observations and test scores, then use AI to generate comprehensive evaluation reports.

### Client Info
Basic profile — name, date of birth, age, sex, diagnosis, notes, and primary concerns.

### Guardian Info
Add and manage guardians (parents, caregivers). For each guardian:
- Name, email, phone, and relationship
- **Invite to Dashboard** — Send an email invitation so the parent can access the Parent Portal

### Clinical Details
Clinical notes, observations, and other provider-specific information.

### Service Details
Configure which services and rates apply to this client.

### Documents
Upload and manage documents related to this client.

### Account History
Activity log showing key events and changes.

### Billing & Charges
View billing history, manage payment methods, and process charges. See [Billing & Payments](/features/billing-payments) for details.

### Access & Permissions
Control which team members and entities can view or edit this client's information.
