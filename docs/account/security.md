# Security

Therativities is designed with security and privacy as a priority.

## Authentication

Your account is secured through Auth0, an industry-leading authentication provider.

## Data protection

- All data is encrypted in transit (TLS/SSL) and at rest
- Role-based access controls ensure users only see what they should
- We use Convex as our backend, which provides enterprise-grade security

## HIPAA

Therativities is designed for HIPAA compliance. See our [HIPAA Compliance page](https://app.therativities.com/hipaa-compliance) for current status and details.
